# How It Works ❔

I'm going to explain how does the platform works in an easy way using scenarios so you can see the full picture and the core idea.

### Scenario 🇦

Person X is sick so he's going to the hospital but wait.. he doesn't have money to pay the bill or the medicine.. he is lucky because the hospital is verified in our platform so the hospital treats him for free at no cost then the hospital sends a **donation request in our platform**, the bill + medicine cost $100 so it includes that price + a photo of the bill: 

1. medicine names 💊
2. quantities 3️⃣
3. costs 💲
4. Hospital name + stamp 🏥
5. Patient fake ID (generated by our platform) #️⃣

**NOT** patient's photo, name, national ID, phone number etc.. why? because we respect patients' privacy.

Now the donation request is public in `/requests` page for the public so donors now can donate to the hospital safely.

#### 🪜 The Flow Summary

Sick patient ➡️ Hospital (verified) ➡️ Treats for free 🆓➡️ Uploads bill (no patient name/face)
    ⬇️
Donation request public 🌐
    ⬇️
Donor sees bill photo 🧾
    ⬇️
Donor pays hospital $100 😌
    ⬇️
Hospital gets money 😊
    ⬇️
Patient already healthy ✅
